Miimi As-saakinah

Miimi As-saakinah Miimi saakinah jechuun qubeen miimi ( م ) sukuuna tahe ykn harakaa tokko malee (fathaa,dommaa,kasraa) alatti yoo dhuftee jechuudha.

Seeroota miimi saakinah

Miimin saakinah seeroota sadi qabdi. Isaanis

  1. idgaama
  2. ikfaa'a
  3. izhaara
jedhaman

Idgaama

Miimin saakina kan taatuu qubeen isii booda dhufu miim ( م ) yoo ta'e qofa dha. Kanaafu miimi saakinah booda miim (م) yoo dhufte idgaama goonee dubbifna jechuudha.

Fkn

Ikfaa'a

Miimi saakinah ikfaa'a kan taatuu yoo qubeen isii booda dhufu baa ( ب ) ta'e dha. Kanaafuu,miimi saakinaah booda baan (ب) yoo dhufte ikfaa'a goonee dubbisnaa.(miimi baa keessatti dhoksinee dubbisnaa.)

Fkn

Izhaara

Miimin saakinah kan izhaara taatuu yoo qubeen isii booda dhufu miimi ( م ) fi baan( ب ) ala tahaniidha. Kana jechuun qubeewwan jiran 28 keessaa lamaanin alatti (م fi ب) miimi saakinah boodatti qubeewwan hafan yoo dhufan izhaara (ifa) goonee dubbisna

Fkn

Irra caalatti:-qubeen miimi saakinaa booda dhufe faa ( ف ) ykn waaw ( و ) yoo ta'an sirritti ifa goonee dubbisna jechuudha.
