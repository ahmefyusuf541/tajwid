Ikfaa'a

Ikfaa'a jechuun dhoksuu jechuu yoo ta'u:- nuuni saakinah fi tanwiiana booda qubeewwan ikfaa'a yoomu dhufan izhaara fi idgaama jiddutti goonee (ifa osoo hin godhinifi, osoo wal keessa hin seensisin dubbisnaa).

Qubeewwan ikfaa'a

Qubeewwan ikfaa'a kan jedhaman qubee izhaara,idgaama fi iqlaaba irraa kan hafani dha. Lakkofsi isaanis 15 .

  1. suaad ( ص )
  2. zaal ( ذ )
  3. s'aa ( ث )
  4. kaaf ( ك )
  5. jiim ( ج )
  6. shiin ( ش )
  7. qaaf ( ق )
  8. siin ( س )
  9. daal ( د )
  10. xua ( ط )
  11. zaa ( ز )
  12. faa ( ف )
  13. taa ( ت )
  14. duaad ( ض )
  15. zuaa ( ظ )
