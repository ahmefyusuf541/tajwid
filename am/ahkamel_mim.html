ሚም አስ-ሳኪናህ ማለት ሚም ፊደል ( م ) ሱኩን ሆና ወይም ያለ ምንም ሀረካ ከ ፈትህ ፣ ዶም እና ከስር ነፃ ሆና ስትመጣ ማለት ነው።

ኢድጋ፟ም

ሚም አስ ሳኪናህ ኢድጋ፟ም የምትሆነው ከሷ ቀጥሎ የመጣው ሀርፍ ሚም ( م ) ሲሆን ነው ። ስለዚህ ሚም አስ ሳኪናህን ከፊት ለፊቷ ሚም ስትመጣ ኢድጋ፟ም አድርገን እናነባታለን።

ምሳሌዎች

ኢኽፋእ

ሚም አስ ሳኪናህ ኢኽፋእ የምትሆነው ከሷ ቀጥሎ የመጣው ሀርፍ ባ ( ب ) ሲሆን ነው ። ስለዚህ ሚም አስ ሳኪናህን ከፊት ለፊቷ ባ ስትመጣ ኢኽፋእ አድርገን እናነባታለን።

ምሳሌዎች

ኢዝሀር

ሚም አስ ሳኪናህ ኢዝሀር የምትሆነው ከሷ ቀጥሎ የመጣው ሀርፍ ከ ባ ( ب ) እና ከ ሚም ( م ) ውጪ ሲሆን ነው ። ስለዚህ ሚም አስ ሳኪናህን ከፊት ለፊቷ ከሁለቱ ፊደሎች ውጪ በመትመጣበት ሰአት ኢዝሀር አድርገን እናነባታለን።

ምሳሌዎች

ነገር ግን ከ ሚም አስ ሳኪናህ ቀጥሎ የመጣው ሀርፍ ፋ ( ف ) ወይም ዋው ( و ) ከሆነ በደንብ ግልፅ ማድረግ ይጠበቅብናል።
