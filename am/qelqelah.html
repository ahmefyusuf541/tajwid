የቀልቀላህ ፊደሎች

  1. ቃፍ ( ق )
  2. ጧ ( ط )
  3. ባ ( ب )
  4. ጂም ( ج )
  5. ዳል ( د )
እነዚህን ፊደሎች ለማስታወስ ቁጥቡ ጀዲን ( قطب جد ) የሚለውን መሸምደድ በቂ ነው። የኢቅላብ ፊደሎች ስኩን ሆነው ሲመጡ እንደ አገባባቸው በሁለት ከፍለን ማየት እንችላለን።
  1. ትንሹ ሱኩኑ አስል/መሰረታዊ ሆና ስትመጣ ወይም መሀል ላይ ስትመጣ
  2. ትለቁ ሱኩኑ በማቆማችን ምክንያት መጤ/አዲስ ሲሆን ወይም መጨረሻ ስትሆን
